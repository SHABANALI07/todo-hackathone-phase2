# Todo Hackathon Phase 2

Welcome to the Todo Hackathon Phase 2 project! This is a comprehensive todo application developed as part of a hackathon challenge.

## Project Overview

This project contains both frontend and backend components for a modern todo application. The architecture follows a microservices approach with separate concerns for the user interface and data management.
